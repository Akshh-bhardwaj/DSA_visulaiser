# 🧠 DSA Visualizer Console (C Language)

This is a **menu-driven Data Structures Visualizer** built using the C programming language.  
It runs in the terminal and allows users to interact with core DSA components like Stack, Queue, and Linked List.

---

## 📌 Features

✅ Visualize and interact with:
- **Stack** (Push, Pop, Display)  
- **Queue** (Enqueue, Dequeue, Display)  
- **Linked List** (Insert at End, Delete by Value, Display)  

🔁 Fully menu-driven for smooth terminal experience.

🧑‍💻 Ideal for:
- Beginners learning Data Structures
- Practice for technical interviews
- Demonstrating DSA knowledge on GitHub
